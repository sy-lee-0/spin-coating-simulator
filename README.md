# Spin Coating Thin-Film Simulator

Chemical Engineering Fluid Mechanics Term Project
Sungkyunkwan University (2026 Spring)

## Description

This project reconstructs the Emslie-Bonner-Peck spin coating theory using:

- Navier-Stokes equations
- Lubrication approximation
- Meyerhofer viscosity model
- RK4 numerical integration

## Features

- Real-time spin coating simulation
- Analytical solution validation
- Design exploration mode
- Thickness evolution visualization
